# 幸福存摺

零時小學校提案，小草書屋的幸福存摺。

> 小草書屋的幸福存摺運作
> - Step 1 孩子們在書屋執行所有任務即可獲得相對應的「金額」
> - Step 2 每周進行一次統計
> - Step 3 填寫於紙本的幸福存摺
> - Step 4 定期審視存摺內的金額 隨時可視實際發生需求提領
> 希望建置一套電子資訊系統，能結合類銀行模式，讓孩子實際感受獲得自身付出所得到的酬勞，進而推廣至全台灣的非營利事業組織兒少陪伴單位。
